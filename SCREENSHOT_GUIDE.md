# Screenshot Guide

This guide will help you capture screenshots for the README.

## Required Screenshots

### 1. Landing Page
- **URL**: `http://localhost:3001/`
- **Filename**: `public/screenshots/landing-page.png`
- **What to show**: Full landing page with hero section and feature cards
- **Browser size**: 1920x1080 or larger
- **Tips**: Make sure the page is fully loaded and animations have settled

### 2. Dashboard - Light Mode
- **URL**: `http://localhost:3001/garden`
- **Filename**: `public/screenshots/dashboard-light.png`
- **What to show**: Main dashboard with stats, recent notes
- **Mode**: Light mode (click sun icon in sidebar if needed)
- **Browser size**: 1920x1080 or larger
- **Tips**: Create a few sample notes first so the dashboard looks populated

### 3. Dashboard - Dark Mode
- **URL**: `http://localhost:3001/garden`
- **Filename**: `public/screenshots/dashboard-dark.png`
- **What to show**: Same as light mode but in dark theme
- **Mode**: Dark mode (click moon icon in sidebar)
- **Browser size**: 1920x1080 or larger
- **Tips**: Same view as light mode to show the beautiful contrast

### 4. AI Chat Interface
- **URL**: `http://localhost:3001/ai/chat`
- **Filename**: `public/screenshots/ai-chat.png`
- **What to show**: Chat interface with a few messages (both user and AI)
- **Mode**: Either light or dark
- **Browser size**: 1920x1080 or larger
- **Tips**: Have a conversation with the AI first, then take the screenshot

## How to Take Screenshots

### On macOS:
1. Press `Cmd + Shift + 4`
2. Press `Spacebar` to capture the entire window
3. Click on the browser window
4. The screenshot will be saved to your Desktop
5. Move it to `public/screenshots/` and rename accordingly

### On Windows:
1. Use Snipping Tool or Snip & Sketch
2. Capture the browser window
3. Save to `public/screenshots/` with the appropriate name

### Using Browser DevTools:
1. Open DevTools (F12)
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows)
3. Type "screenshot" and select "Capture full size screenshot"
4. Save and move to `public/screenshots/`

## After Taking Screenshots

Once all screenshots are saved in `public/screenshots/`, they will automatically appear in the README!
